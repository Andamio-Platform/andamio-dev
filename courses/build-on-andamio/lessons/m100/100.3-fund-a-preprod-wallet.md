# Fund a preprod wallet

## Before we start

Every Andamio write operation is a Cardano transaction, and transactions cost ADA. On preprod, ADA is free from a faucet.

Most operations in this course — creating courses, submitting evidence, claiming credentials — are on-chain transactions. The exceptions are course module content and project task drafts, which are written off-chain before any transaction commits them. Module 400 covers the transaction mechanism; Modules 500 and 600 show the draft-then-commit workflow.

## You'll need a preprod address

The faucet sends ADA to an `addr_test1...` address on preprod. If you already have one, skip ahead.

- **Browser wallet.** Install [Eternl](https://eternl.io) or another CIP-30 wallet, switch to **Preprod**, create an account, copy the receiving address.
- **CLI keys.** If you have `cardano-cli`:

  ```bash
  cardano-cli address key-gen \
    --verification-key-file payment.vkey \
    --signing-key-file payment.skey

  cardano-cli address build \
    --payment-verification-key-file payment.vkey \
    --testnet-magic 1 \
    --out-file payment.addr
  ```

  Keep the `.skey` file — you'll use it in M200.3 for headless login.

## The faucet

1. Open [docs.cardano.org/cardano-testnets/tools/faucet](https://docs.cardano.org/cardano-testnets/tools/faucet)
2. Select **Preprod**
3. Paste your `addr_test1...` address
4. Complete the ReCAPTCHA, click **Request Funds**

The page returns a transaction hash. Funds land within 1–2 minutes. Current limits: 10,000 test ADA per request, one request per 24 hours per IP. The page is authoritative on current numbers.

## Verify

- **Browser wallet:** open it on Preprod, wait for the balance to update.
- **Block explorer:** paste your address into [preprod.cardanoscan.io](https://preprod.cardanoscan.io) and check the UTxO section.

Done when your address shows a nonzero balance.

## What the faucet actually is

The testnet faucet is maintained by Input Output Global (IOG), the company that built Cardano. It holds test ADA generated at network genesis and hands it out rate-limited by IP. It has no relationship to Andamio — every Cardano developer uses the same faucet.

Preprod ADA has no real-world value. Mainnet ADA does. The two networks are isolated. Don't mix them up when you ship to mainnet.

## Your turn

Go to the faucet, request funds, verify your balance. If you're with an agent, it can't click the ReCAPTCHA — but it can help verify the balance afterward.

Done when your preprod address shows a nonzero balance.

## What you just did

You have a funded preprod wallet. The faucet is standard Cardano infrastructure, not an Andamio resource — the wallet works with every Cardano tool, not just this course. The wallet is yours. The ADA is yours. Both outlive this lesson.
